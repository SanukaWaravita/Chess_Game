# Stage 1: Chess board UI

- Goal: show an 8×8 board.

- Concepts:
    - Board coordinates: files a-h, ranks 1-8
    - 2D array board representation
    - Light/dark square coloring
    - Render pieces on squares

```javascript

  ["p","p","p","p","p","p","p","p"],
  [null,null,null,null,null,null,null,null],
  ...
]
```
